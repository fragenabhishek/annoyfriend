# Automated Typing Script

This Python script uses the `pyautogui` library to automate typing messages with randomly selected animals.

## Usage

1. Clone the repository:
   ```bash
   git clone https://github.com/your_username/automated-typing-script.git
Navigate to the project directory:


cd automated-typing-script
Install the required libraries:


pip install -r requirements.txt
Run the script:


python typing_script.py

Customization
You can customize the behavior of the script by modifying the following parameters:

Message: Change the message to be typed by editing the custom_message variable in the script.
Animals: Edit the list of animals by modifying the custom_animals variable in the script.
Iterations: Adjust the number of iterations by changing the value of the custom_iterations variable in the script.
Delay: Set the delay between each iteration (in seconds) by modifying the custom_delay variable in the script.
